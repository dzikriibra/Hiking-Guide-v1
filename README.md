# 🏔️ Panduan Mendaki Gunung (Untuk Pemula)

> **Submission Tugas Akhir: Belajar Dasar Pemrograman Web - Dicoding Indonesia**

Sebuah landing page edukatif yang dirancang khusus untuk memberikan panduan komprehensif bagi pendaki pemula. Project ini mencakup persiapan fisik, manajemen perlengkapan, hingga teknik pendakian yang aman berdasarkan riset dan literatur pendakian.

---

## 🎯 Project Overview

Dalam beberapa tahun terakhir, tren mendaki gunung meningkat drastis. Namun, banyak pendaki pemula yang melakukan perjalanan tanpa persiapan memadai. Website ini hadir untuk:

- **Edukasi Risiko:** Menjelaskan bahaya hipotermia dan AMS (_Acute Mountain Sickness_).
- **Manajemen Logistik:** Memberikan checklist peralatan wajib dan opsional.
- **Optimasi Teknik:** Panduan _packing_ carrier dan teknik berjalan yang efisien.

## 🚀 Tech Stack

Project ini dibangun menggunakan teknologi web dasar untuk memastikan performa yang ringan dan responsif:

- **HTML5:** Struktur semantik (Article, Section, Aside, Figure).
- **CSS3:** Custom styling dengan sistem Grid dan Flexbox untuk layouting.
- **Font Awesome:** Library icon untuk elemen sosial media dan antarmuka.
- **Google Fonts:** Menggunakan font _Quicksand_ dan _Shadows Into Light_ untuk tipografi yang modern.
